# RestApp
A powerful Java RESTful application based on Jersey using Mentabean ORM

Feel free to use this project as template for your RESTful web application.
Enjoy it!

## Technology Stack
- Jersey (JAX-RS)
- Mentabean (ORM)
- MentaContainer (IoC)
- HikariCP (Connection Pool)
- Jackson (JSON provider)
- JodaTime (Date/Time)
- Grizzly (Web Server)
- H2 (JUnit tests)
